# まちから 仕入・販売管理ダッシュボード

ブラウザだけで動く仕入・販売管理ダッシュボードです。サーバーやインストールは不要です。

## 開き方

`index.html` をダブルクリックしてブラウザ(Chrome / Edge / Safari)で開くだけです。
フォルダごと移動・コピーしても動きます(4ファイルセットで保管してください)。

| ファイル | 役割 |
|---|---|
| `index.html` | ダッシュボード本体 |
| `data.js` | 初期データ(販売 2023-01〜2025-12 / 漁協仕入 2024-01〜2025-12) |
| `pdf.min.js` / `pdf.worker.min.js` | PDF読み取りライブラリ(オフライン動作用) |

## 画面構成

- **概要** — 販売額(税抜)・粗利・漁協仕入額のKPIと月別推移、年月別一覧表
- **仕入分析** — 魚種群別・商品別・月別の仕入額、単価、数量(kg)
- **販売分析** — 商品群別・商品別・顧客別・月別の販売額と粗利
- **明細** — 販売・仕入の生明細。商品名クリックで月別推移を表示
- **データ管理** — 新しいデータの取り込み

共通操作: 上部の期間ボタン(年/直近12ヶ月/任意の月範囲)と商品検索は全タブに効きます。
グラフの棒をクリックすると絞り込み、表の見出しクリックで並べ替え、各表はCSVダウンロード可能です。

## データの更新方法(データ管理タブ)

- **仕入**: 漁協システムの CSV (`DH8A_K*.CSV` など、Shift-JIS) をドロップ。複数ファイル同時可。
- **販売**: 販売システムの「売上明細一覧」PDF をドロップ。ページ数が多いと数分かかります。

取り込んだファイルに含まれる**月の既存データは丸ごと置き換え**られるので、
同じ月を何度取り込んでも二重計上になりません。

追加データはブラウザ(localStorage)に保存され、次回開いたときも自動で反映されます。
「追加データを全て削除」ボタンで初期データの状態に戻せます。

## 商品群の分類

商品名のキーワードによる自動分類です(データ管理タブに一覧表示)。
分類ルールは `index.html` 内の `SALE_RULES` / `PUR_RULES` で変更できます。

## 注意

- 粗利額は販売システム出力値をそのまま使用しています。原価未登録の商品は粗利=販売額になるため、全体の粗利率は実態より高く出ます。
- 漁協仕入CSVの「消費税」行は商品集計から除外しています(金額はすべて税抜ベース)。
- 初期データの再生成や分類ルールの調整が必要になったら Claude に依頼してください。
